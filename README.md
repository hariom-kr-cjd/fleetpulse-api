# FleetPulse API

REST API for Fleet & Driver Management System.

## Tech Stack
- Node.js 18+ / Express.js / TypeScript
- MongoDB / Mongoose
- JWT Authentication with RBAC
- Jest + Supertest for testing

## Setup
```bash
cp .env.example .env
npm install
npm run dev
```

## Test
```bash
npm test
```

## Seed Data
```bash
npm run seed
```

## API Base URL
`http://localhost:3000/api/v1`
